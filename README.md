## About Project

Test task for backend Laravel developer

## Initialization

### Prepare:
- ```git clone```
- ```composer install```
- ```npm install```
- ```cp .env.example .env```

### ENV:
- ```APP_URL``` - main url of website
- ```DB_``` - DB connection

### Main:
- ```php artisan key:generate```
- ```php artisan storage:link```
- ```php artisan migrate```
- ```php artisan adminlte:install```
- ```php artisan optimize```
